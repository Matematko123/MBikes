# MBikes
Bike store app using MERN stack

<h1> <b>Features</b> </h1>
<h2> BACKEND </h2> 
<ul>
  <li> Full REST API </li>
  <li> CRUD - MongoDB </li>
  <li> Authentication system </li>
</ul>

<h2> FRONTEND </h2> 
<ul>
  <li> Routing </li>
  <li> Cart page </li>
  <li> Redux for state managment </li>
  <li> Admin panel / CMS </li>
  <li> Login / Register </li>
  <li> Add to cart </li>
  <li> Stripe api for payment </li>
</ul>
